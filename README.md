# cura-cli

A smart skill manager for your repositories. `cura` analyzes your codebase and suggests the right skills to add context for your AI coding workflows.

## What it does

- **Scans your repo** — detects languages, frameworks, and project structure
- **Suggests skills** — uses LLMgo to recommend 3–5 relevant skill files
- **Manages skills** — add, remove, and track skills per repo
- **Tracks token usage** — monitors how many tokens you're spending per repo with hard limits
